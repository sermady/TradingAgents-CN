# TradingAgents分析师团队提示词分析报告

## 概述

本文档详细分析了TradingAgents项目中各个分析师的提示词配置，包括系统消息、工具调用要求和输出格式等。

## 分析师团队组成

TradingAgents包含以下分析师：

1. **市场分析师 (market_analyst)** - 技术分析专家
2. **基本面分析师 (fundamentals_analyst)** - 财务分析专家  
3. **新闻分析师 (news_analyst)** - 新闻事件分析专家
4. **社交媒体分析师 (social_media_analyst)** - 情绪分析专家
5. **中国市场分析师 (china_market_analyst)** - 中国市场专家
6. **看涨研究员 (bull_researcher)** - 投资辩论中的看涨方
7. **看跌研究员 (bear_researcher)** - 投资辩论中的看跌方

## 详细提示词分析

### 1. 市场分析师 (Market Analyst)

**核心定位**: 专业股票技术分析师

**主要职责**:
- 技术指标分析（MA、MACD、RSI、布林带等）
- 价格趋势分析
- 成交量分析
- 投资建议

**工具调用**:
- 统一使用 `get_stock_market_data_unified` 工具
- 自动识别股票类型（A股/港股/美股）
- 最大工具调用次数：3次

**输出格式要求**:
```
## [CHART] 股票基本信息
## [CHART-UP] 技术指标分析  
## [CHART-DOWN] 价格趋势分析
## [IDEA] 投资建议
```

**特殊要求**:
- 必须使用工具获取真实数据
- 不能重复调用工具
- 基于工具数据生成完整技术分析报告
- 价格数据使用相应货币单位

### 2. 基本面分析师 (Fundamentals Analyst)

**核心定位**: 专业股票基本面分析师

**主要职责**:
- 财务数据分析
- 估值指标计算（PE、PB、PEG等）
- 合理价位区间分析
- 基于基本面的目标价位建议

**工具调用**:
- 统一使用 `get_stock_fundamentals_unified` 工具
- 自动获取价格数据和基本面数据
- 最大工具调用次数：1次

**输出格式要求**:
- 基于真实数据进行深度基本面分析
- 计算并提供合理价位区间
- 分析当前股价是否被低估或高估
- 提供基于基本面的目标价位建议

**严格禁止**:
- 不允许说"我将调用工具"
- 不允许假设任何数据
- 不允许编造公司信息

### 3. 新闻分析师 (News Analyst)

**核心定位**: 专业财经新闻分析师

**主要职责**:
- 获取和分析最新实时新闻
- 评估新闻事件的紧急程度和市场影响
- 识别可能影响股价的关键信息
- 分析新闻的时效性和可靠性

**工具调用**:
- 统一使用 `get_stock_news_unified` 工具
- 优先获取15-30分钟内的新闻
- 最大工具调用次数：3次

**分析要点**:
- 新闻的时效性（发布时间距离现在多久）
- 新闻的可信度（来源权威性）
- 市场影响程度（对股价的潜在影响）
- 投资者情绪变化（正面/负面/中性）

**重点关注新闻类型**:
- 财报发布和业绩指导
- 重大合作和并购消息
- 政策变化和监管动态
- 突发事件和危机管理
- 行业趋势和技术突破
- 管理层变动和战略调整

### 4. 社交媒体分析师 (Social Media Analyst)

**核心定位**: 中国市场社交媒体和投资情绪分析师

**主要职责**:
- 分析中国主要财经平台的投资者情绪
- 监控财经媒体和新闻对股票的报道倾向
- 识别影响股价的热点事件和市场传言
- 评估散户与机构投资者的观点差异

**工具调用**:
- 统一使用 `get_stock_sentiment_unified` 工具
- 最大工具调用次数：3次

**重点关注平台**:
- 财经新闻：财联社、新浪财经、东方财富、腾讯财经
- 投资社区：雪球、东方财富股吧、同花顺
- 社交媒体：微博财经大V、知乎投资话题
- 专业分析：各大券商研报、财经自媒体

**情绪影响分析要求**:
- 量化投资者情绪强度（乐观/悲观程度）和情绪变化趋势
- 评估情绪变化对短期市场反应的影响（1-5天）
- 分析散户情绪与市场走势的相关性
- 识别情绪极端点和可能的情绪反转信号
- 提供基于情绪分析的市场预期和投资建议

### 5. 中国市场分析师 (China Market Analyst)

**核心定位**: 中国A股市场专家

**主要职责**:
- 专门分析中国A股市场特点
- 结合中国市场政策环境进行分析
- 考虑中国投资者行为特征

**工具调用**:
- 使用专门的中国市场数据工具
- 针对A股市场的特殊分析

### 6. 看涨研究员 (Bull Researcher)

**核心定位**: 看涨分析师，负责建立强有力的投资论证

**主要职责**:
- 为股票投资建立强有力的看涨论证
- 强调机遇、优势和积极指标
- 利用数据反驳看跌论点
- 提供买入或持有的建议

**分析重点**:
- 机遇和优势：突出增长潜力、市场机会和积极因素
- 积极指标：强调财务健康、市场表现和创新优势
- 市场地位：分析竞争优势和护城河
- 未来前景：讨论增长策略和发展计划

**语言要求**:
- 使用中文
- 强调积极的投资建议
- 基于数据和分析支持观点

### 7. 看跌研究员 (Bear Researcher)

**核心定位**: 看跌分析师，负责论证不投资的理由

**主要职责**:
- 论证不投资股票的理由
- 强调风险、挑战和负面指标
- 利用数据支持看跌立场
- 提供卖出或避免的建议

**分析重点**:
- 风险和挑战：突出市场饱和、财务不稳定或宏观经济威胁
- 竞争劣势：强调市场地位较弱、创新下降或来自竞争对手威胁
- 负面指标：使用财务数据、市场趋势或最近不利消息的证据

## 统一特性

### 1. 工具调用机制
- 所有分析师都使用统一的工具接口
- 自动识别股票类型（A股/港股/美股）
- 防止无限循环的工具调用限制
- 基于真实数据进行分析

### 2. 多语言支持
- 所有提示词要求使用中文分析
- 投资建议使用中文（买入/持有/卖出）
- 货币单位根据市场自动适配

### 3. 数据验证
- 严格禁止假设或编造数据
- 必须使用工具获取真实数据
- 基于真实数据进行专业分析

### 4. 格式化输出
- 统一的markdown格式要求
- 专业的分析报告结构
- 清晰的投资建议标注

### 5. 记忆机制
- 支持历史记忆功能
- 基于过往分析经验改进当前分析
- 防止重复错误

## 工具配置

### 统一工具列表
- `get_stock_market_data_unified` - 市场数据统一接口
- `get_stock_fundamentals_unified` - 基本面数据统一接口  
- `get_stock_news_unified` - 新闻数据统一接口
- `get_stock_sentiment_unified` - 情绪数据统一接口

### 股票类型识别
- A股：中国大陆股票市场
- 港股：香港股票市场
- 美股：美国股票市场

每个工具都会自动识别股票类型并调用相应的数据源。

## 质量保证

### 1. 死循环防护
- 每个分析师都有工具调用计数器
- 达到最大调用次数后强制结束
- 防止LLM陷入无限调用循环

### 2. 数据验证
- 所有分析师必须使用工具获取真实数据
- 不允许假设或编造信息
- 基于真实数据进行专业分析

### 3. 输出质量
- 统一的专业格式要求
- 详细的分析报告结构
- 明确的投资建议标注

### 4. 错误处理
- 完善的降级方案
- 工具调用失败的备用处理
- 详细的日志记录和错误追踪

## 总结

TradingAgents的分析师团队提示词设计体现了以下特点：

1. **专业化分工**: 每个分析师都有明确的专业领域和职责
2. **工具驱动**: 强调基于真实数据的分析
3. **格式统一**: 规范化的输出格式要求
4. **多市场支持**: 自动适配不同股票市场
5. **质量保证**: 多重机制确保分析质量
6. **中文优化**: 专门针对中文市场和用户优化

这些提示词配置确保了分析师团队能够提供专业、一致、高质量的股票分析服务。
